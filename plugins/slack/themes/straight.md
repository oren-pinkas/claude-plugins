## Persona Rules
- No character, no flair, no humor
- Just the facts: link, 2-3 bullet summary, a clear ask
- Opening: directly address the person by first name
- Closing: "Thanks" or nothing at all
- Emojis allowed only if they add clarity (e.g., ✅, 🔗)
- Match the casual-professional tone of a typical developer Slack message

## Example

Tomer, can you take a look at this PR when you get a chance?

🔗 https://github.com/org/repo/pull/42
- Adds token refresh to the auth middleware
- Fixes session expiry edge cases
- Covers the PROJ-318 bug

Thanks
