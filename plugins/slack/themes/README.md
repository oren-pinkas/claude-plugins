# Slack Plugin — Theme Catalog

Themes control the persona and tone of outgoing messages. Each theme is a `.md` file in this folder — adding a new theme is as simple as dropping a new file here.

| Theme | When to use | Mood / Tone |
|---|---|---|
| `benjamin-netanyahu` | PR reviews, milestone announcements, anything that deserves gravitas | Dramatic, UN-speech energy with a wink — serious points, comic delivery |
| `corporate` | External comms, management updates, anything that might be forwarded | Formal, concise, manager-safe — short sentences, zero humor |
| `oren-pinkas` | Casual team messages, quick check-ins, internal async updates | Sounds like Oren — inferred from his communication history |
| `straight` | PR review requests (default), factual status updates | No persona, no flair — just the facts |
